# Koncert App

Web aplikacija za upravljanje koncertima.

## Tech Stack
- **Backend:** Java 21, Spring Boot 3.x, Spring Data JPA, MySQL
- **Frontend:** Angular, TypeScript

## Pokretanje

### Backend
1. Kopiraj `backend/src/main/resources/application.properties.example` u `application.properties`
2. Unesi svoje MySQL kredencijale
3. Pokreni: `mvn spring-boot:run`

### Frontend
1. `cd frontend`
2. `npm install`
3. `ng serve`
